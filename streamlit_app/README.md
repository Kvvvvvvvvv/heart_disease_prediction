# Heart Disease Prediction - Streamlit App

This is a deployment-ready version of the heart disease prediction application designed specifically for Streamlit Sharing.

## Features

- Predicts heart disease risk based on patient medical attributes
- Interactive UI with Streamlit
- Machine learning model powered by scikit-learn
- Risk probability scoring

## How to Use

The application is designed to work with Streamlit Sharing:

1. The model files (`heart_disease_model.pkl` and `feature_names.json`) should be in the project root or data directory
2. The app will automatically locate these files
3. Enter patient information in the form
4. Click "Predict Heart Disease" to get results

## Requirements

All requirements are specified in `requirements.txt`.

## Note

This application is for educational purposes only and should not be used for actual medical diagnosis.
